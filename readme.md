## Getting Started

### Prerequisites

Before you begin, ensure you have the following dependencies installed on your system:

- Node.js: [Download Node.js](https://nodejs.org/)

### Installation

1. Clone this repository to your local machine:

   ```bash
   git clone git@github.com:Peaky00/employeetracker.git
cd employee-tracker
npm install

## Usage
Starting the Application
To start the Employee Tracker CLI Application, run the following command: node server.js

Options and Actions
Upon starting the application, you will be presented with the following options:

View all departments
View all roles
View all employees
Add a department
Add a role
Add an employee
Update an employee role
Exit
